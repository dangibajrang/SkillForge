# Project Overview

## Product Information

**Product Name:** SkillForge – Learning & Assessment Platform

**Product Goal (North Star):** Build a scalable, secure, microservices-based learning platform with event-driven architecture, capable of handling authentication, content delivery, assessments, and notifications.

## Methodology

- **Framework:** Agile (Scrum)
- **Total Sprints:** 6
- **Sprint Duration:** 1 week each
- **Sprint Cadence:** End of every sprint → demo + refactor

## Core Objectives

1. **Scalability:** Microservices architecture to handle growth
2. **Security:** OAuth-based authentication with JWT authorization
3. **Performance:** CDN-based media delivery for optimal user experience
4. **Reliability:** Event-driven architecture for decoupled, resilient systems
5. **User Experience:** Modern frontend with seamless backend integration

## Success Criteria

- ✅ Microservices architecture implemented
- ✅ OAuth + JWT security in place
- ✅ Event-driven system operational
- ✅ CDN-based media delivery functional
- ✅ Real frontend-backend integration complete
- ✅ Production-ready with proper error handling and logging
- ✅ Comprehensive documentation for onboarding

## Target Users

- **Learners:** Students taking courses and assessments
- **Instructors:** Content creators managing courses
- **Administrators:** System managers overseeing platform operations

## Key Principles

Following Big Tech best practices:
- Build incrementally
- Keep services independent
- Prioritize clarity over speed
- Focus on why, not just what
